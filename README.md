# Termi Protocol

**The control room for your AI coding agents.**

Termi Protocol turns terminal-based AI coding agents into a live 3D workspace. Run Claude Code, Codex, Gemini CLI, Grok CLI, Copilot, Aider, opencode, Ollama or any compatible CLI agent on your own machine, then watch what they read, write and execute in real time.

Download the desktop app:

Termi Protocol is available for macOS, Windows and Linux. Bring your own AI coding CLI and API keys. Your agents run locally, your code stays on your machine, and Termi visualizes the work in a cozy Three.js workspace.

## Features

- **Live 3D agent workspace**: built with Three.js so every file read, file edit and command becomes visible.
- **Bring your own agent**: Claude Code, Codex, Gemini CLI, Grok CLI, Copilot, Aider, opencode, Ollama, Groq, Amazon Q and other CLI agents.
- **Command Center**: real terminal, task board, checkpoint history, memory, token usage and cost in one panel.
- **Agent task status**: see what each agent is doing now, what is blocked, what is done and what needs approval.
- **Checkpoints and rewind**: recover from a bad edit or runaway command without losing the whole session.
- **Multi-agent control**: file locks, task handoff and status boards help multiple agents work without collisions.
- **Gamified workflow**: XP, leagues, leaderboard, cozy room customization, desk tools, toys and pets.
- **Local-first by design**: the app orchestrates your own tools on your machine.

## Why Termi Protocol?

AI coding agents are powerful, but they are often opaque: a scrolling terminal, hidden context, unclear state and risky edits. Termi Protocol gives agents a body, a room and a visible workflow so developers can inspect progress without giving up control.

Use it when you care about:

- AI agent observability
- Claude Code workflows
- Gemini CLI and Codex workflows
- LLM task automation
- local and open-source model workflows through compatible CLIs
- agent status dashboards
- coding agent checkpoints
- cozy productivity games
- gamified developer tools
